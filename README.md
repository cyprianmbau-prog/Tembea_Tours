# Tembea_Tours
**Description**
This project is a website page build using HMTL,CSS and Javascript.
It displays several  travel destinations, images and some input forms for users to book their travel destinations which they are interested in.
The site is designed to:
-displays attractive tour destinations
-provide booking forms for users to book
-detailed tour sites with locations where they are found
-provide feedbacks according to users input
**Technologies used**
+HTML
+CSS
+Javascript
